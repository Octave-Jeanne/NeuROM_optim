# NeuROM-optim
A pytorch-based package to solve topology optimisation problems for (non) linear 2D magnetostatics problems.

## Installation

1. First create a conda env running `conda create -n neurom-optim-env python 3.13 -y`
2. Activate the env `conda activate neurom-optim-env`
3. Then clone the project `git clone https://github.com/Octave-Jeanne/NeuROM_optim`
4. Go in the cloned directory `cd NeuROM_optim`
5. Install the library in editable mode `pip install -e .`
